# CHANGELOG

このプロジェクトの特筆すべき変更を記録する。形式は [Keep a Changelog](https://keepachangelog.com/ja/1.1.0/) に準拠。
変更を merge するたびに `Unreleased` へ 1 行追記し、節目で日付付きセクションへ切り出す（[CONTRIBUTING.md](./CONTRIBUTING.md)）。

## [Unreleased]

### Added
- AI 開発ハーネスを導入（`docs/` 配下のドキュメント一式・AI_RULES・CONTRIBUTING・CHANGELOG）
- グローバルナビ（下部タブバー `BottomNav`）を全ページに追加し、アプリ間をホーム経由せず移動可能に
- 共通コンポーネント `ConfirmDialog` / `EmptyState` / `Toast` / `OfflineBanner` を追加
- オフライン検知バナー（リアルタイム同期の状態を明示）
- 家族参加フローに招待リンク貼り付け欄を追加（`GroupSetup`）

### Changed
- 買い物リストの削除を確認ダイアログ経由に変更（従来は確認なしで即削除）
- 削除確認 UI を全ページで `ConfirmDialog` に統一（Places の編集モーダルの即削除も確認付きに）
- 戻るボタンを全ページで 🏠 アイコン・円形 40px・`aria-label` 付きに統一（Inventory / Travel の `←` を置換）
- 主要な操作系のタップ領域を 40〜52px 以上へ拡大（戻る・リスト削除の当たり判定・ダイアログ・タブ）
- お気に入り更新失敗を無言のロールバックからトースト通知に変更
- お出かけリストの操作バーを整理：検索＋「絞り込み」トグル＋ビュー切替の 1 行に集約し、カテゴリ・都道府県・タグ・範囲は折りたたみパネルへ。有効件数バッジと一括解除を追加
- 曜日ヘッダ・「近日公開」バッジの文字サイズを可読性向上のため微増
- スケジュールの「勤務入力モード」をヘッダーの「⋯」メニューに格納し、通常時のヘッダーを簡素化（モード中は明示的な「勤務モード終了」を表示）
- Web フォントの読み込みを CSS `@import`（レンダリングブロッキング）から `index.html` の `preconnect` + `<link>` に移設し初回描画を改善
- 下部ナビ（BottomNav）を `position: fixed` で画面下部に常時固定（従来はレイアウトによりスクロールで動く場合があった）。各ページのスクロール領域はナビ高さ分の余白を確保
